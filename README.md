## 🌐 Understanding the Connection

When you type a website address, your browser sends a request. Internet providers inspect these requests to see if they should block them. EasySNI intercepts these requests before they leave your computer. It scrambles the packet headers. The provider sees a incomplete request and allows it to pass. The web server at the other end receives the full request and displays the site. This happens in milliseconds.
